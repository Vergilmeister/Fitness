Security

🔒 Secure & Private

Your data. Your rules. Always.

Security is built into every layer of FitLife. From password hashing to token validation and server-side input checking, your data is protected end-to-end. You never have to think about it — it just works.

✦ What's Included
Everything this feature does
🔑

JWT Authentication

Secure JSON Web Tokens with a 7-day expiry keep sessions safe and stateless.

🛡️

bcrypt Hashing

Passwords are hashed with bcrypt (salt rounds: 10) — never stored in plain text.

✅

Input Validation

All API routes validate inputs server-side using express-validator.

🚫

Route Protection

Every private endpoint requires a valid token — no exceptions.

👤

Ownership Checks

Users can only access, edit, or delete their own workouts and suggestions.

🔐

Env-Based Secrets

JWT secret and API keys are stored in .env — never committed to source code.

✦ How To Use
Get started in minutes
1 Register — your password is hashed before it ever hits the database
2 Log in — receive a secure JWT token stored in your browser
3 Every API call includes your token in the Authorization header
4 Tokens expire after 7 days, requiring a fresh login
